# Xcode File Templates

## Installation

`git clone https://github.com/eastsss/xcode-file-templates.git ~/Library/Developer/Xcode/Templates/File\ Templates/eastsss/`

## Update

`cd ~/Library/Developer/Xcode/Templates/File\ Templates/eastsss/`
`git pull`
